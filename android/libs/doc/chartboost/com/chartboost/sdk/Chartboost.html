      • setImpressionsUseActivities

        public static void setImpressionsUseActivities(boolean impressionsUseActivities)
        Important: Call this method in your activity's onCreate() method before you call Chartboost.onCreate().
        Set whether or not to launch interstitials and the More Apps page in their own activity. This is required for videos to be compatible with some 3rd party frameworks and other apps that use SurfaceViews.

        Please use care, as the onPause-onStop/onStart-onResume cycle will be called for the originating activity when showing and hiding the impressions. If false (default behavior), impressions will be displayed in a view above the other content views in your host activity.
