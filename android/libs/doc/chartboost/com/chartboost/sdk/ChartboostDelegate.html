        public boolean shouldRequestInterstitial(java.lang.String location)
        Called before requesting an interstitial via the Chartboost API server.

        Implement to control if the Charboost SDK should fetch data from the Chartboost API servers for the given CBLocation}. This is evaluated if the Chartboost.showInterstitial(String)} orChartboost.cacheInterstitial(String)} are called. If true is returned the operation will proceed, if false, then theoperation is treated as a no-op.

        Specified by:
        shouldRequestInterstitial in interface com.chartboost.sdk.AbstractChartboostDelegate
        Parameters:
        location - The location for the Chartboost impression type.
        Returns:
